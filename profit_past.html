Project Details

We developed a hybrid trading system that combines predictive machine learning with a reinforcement learning agent capable of reacting to real-time news sentiment. By utilizing a sliding-window approach to constantly adapt to market changes, our model significantly outperformed standard buy-and-hold strategies in tests across major tech stocks mainly NVIDIA.

Project Overview

Objective

The goal of this project was to build an automated trading agent that adapts to changing market conditions by combining predictive Machine Learning (ML) with Reinforcement Learning (RL) and real-time news sentiment analysis.

Core Methodology

Hybrid Architecture: The system uses a "one-step-ahead" return forecaster (trained using models like LSTM, XGBoost, or Random Forest) to predict short-term price movements. An RL agent (Q-Learning) then uses these predictions, along with current market data and sentiment scores, to make Buy, Sell, or Hold decisions.

Sliding Window Mechanism: To prevent the model from becoming obsolete, it employs a dynamic sliding window. The stock prediction model is retrained on the last 30 days of data, while the RL agent updates weekly, ensuring the strategy remains responsive to recent market shifts.

Sentiment Integration: The model processes news headlines to generate sentiment scores (Positive/Negative/Neutral), which are fed directly into the RL agent's state space to influence trading decisions.
