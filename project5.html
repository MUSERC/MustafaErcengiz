Project Overview

The dataset contains various engine sensor readings including RPM, lubricant oil pressure, fuel pressure, coolant pressure, lubricant oil temperature, and coolant temperature. The target variable is "Engine Condition" which is binary (1 for working, 0 for not working/dead).

Technical Implementation

The project was implemented using Python with libraries including pandas, scikit-learn, XGBoost, seaborn, and matplotlib. Data preprocessing included scaling features and handling outliers to improve model performance.
