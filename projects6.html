Mobile Game Churn Prediction

About Projects

Project Details

This project focuses on predicting user churn in the mobile gaming industry using behavioral and engagement data collected within the first week. By training machine learning models to predict churn, developers can proactively implement retention strategies and improve monetization.

Project Overview

Based on a real-world dataset from a mobile puzzle game, this project includes extensive feature engineering, model comparison (Logistic Regression, Random Forest, XGBoost, and more), and a final model deployment.
