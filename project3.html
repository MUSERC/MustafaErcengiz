Project Details

Fully functional end-to-end Platform web app that detects anomalies in plants with a convolutional neural network deployed on the backend and analyzes the possible causes. The app also connects users with plant health providers or bio-recycle firms if any anomaly is detected.

I developed the CNN model and created backend functions, integrating them with ChatGPT's API to increase the app's responsiveness.
